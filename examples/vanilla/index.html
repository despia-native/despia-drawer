GitHub npm Despia

Web Component · iOS-style drawer

A bottom sheet that behaves like it belongs on the device.

Native scroll snap, named detents, keyboard-aware forms, stack layering, and PWA haptics in one small custom element.

Designed for the rough edges.

This page is intentionally plain so you can see what matters: keyboard focus, content scrolling, backdrop behavior, detent changes, and click-through surfaces behind the sheet.

Motion CSS scroll snap with native browser momentum. No JavaScript drag loop.
Keyboard Uses `interactive-widget=resizes-content` when available, with a focused fallback lock.
Inputs Text, select, date, time, file, textarea, autofill, and long-form inner scrolling.
Stacking Opening a second drawer layers it above the previous one and locks the older sheet behind it.
